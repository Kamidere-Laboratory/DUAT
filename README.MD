## Discord Users Activity Tracker [DUAT]
Track users activity on guild specified in process enviromental  
`DSC_TOKEN=TOKEN`  
`DSC_GUILD=GUILDID`  

# Features
Track creating/editing/deleting messeges  
Track voice joining/leaving
